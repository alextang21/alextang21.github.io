I created this game in the summer of 2020. Excited to take computer science classes, I used video tutorials and documentation to create this game with Unity and I published it to the web using GitHub Pages. I utilized Unity's game development platform, C#, and basic HTML and CSS knowledge to create and publish this game.

The creation of this game inspired me to keep learning new concepts and to try to make other games. Through this experience I learned more about my strengths and weaknesses as a developer. One of my strengths being my ability to pick up new syntax and environments quickly, and one of the weaknesses being the difficulty I find in retaining the information. I found myself looking back at the documentation multiple times before I was able to use methods off the top of my head. This is a skill that I have improved on over the years as I progress through courses and work experience.
